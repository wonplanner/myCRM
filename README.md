
# 🛡️ 인슈어플래너 CRM (Insure Planner)

보험 설계사를 위한 iOS 스타일의 프리미엄 고객 관리(CRM) 솔루션입니다. 복잡한 설치 없이 브라우저에서 즉시 고객 데이터를 관리하고 AI 인사이트를 얻으세요.

## 🚀 핵심 가치
- **Privacy First**: 모든 데이터는 설계사의 기기에만 저장됩니다. (Local Storage & Cloud Sync)
- **AI Intelligence**: Google Gemini API를 활용하여 매일 아침 맞춤형 업무 리포트를 제공합니다.
- **Visual Networking**: 고객 간의 복잡한 관계를 한눈에 파악할 수 있는 인맥 지도를 지원합니다.

## ✨ 주요 기능
- **스마트 대시보드**: AI 비즈니스 인사이트 및 개인 업무 리마인더.
- **정밀 고객 필터링**: 유지(Active), 해지(Cancelled), 잠재(Prospect) 상태별 원클릭 필터.
- **인맥 지도**: 가족, 지인, 추천인 관계의 시각적 연결 및 즉시 이동 기능.
- **단체 발송 최적화**: iOS/Android 기기 특성에 맞춘 자동 구분자 적용 단체 SMS 발송.
- **데이터 센터**: 엑셀(CSV) 백업 및 PDF 리스트 출력 지원.

## 🛠️ 기술적 특징
- **Frontend**: React 19, Tailwind CSS, Lucide Icons.
- **AI Engine**: Google Gemini-3-Flash-Preview.
- **Security**: AES 기반 암호화 구조 및 로컬 우선 저장 방식.

## 📥 시작하기 (개발자용)
1. 환경 변수 설정: `.env` 파일에 `API_KEY` (Gemini)를 등록합니다.
2. 의존성 설치: `npm install`
3. 실행: `npm start`
